# LeaderF-bookmark

This Plugin use LeaderF to navigate the bookmark.

# Usage

```
:LeaderfBookmark
```
or
```
:Leaderf bookmark
```

### Commands

#### Add bookmark

```
:BookmarkAdd {name} {path}
```

or

```
:BookmarkAddHere [{name}]
```

or

`N` in normal mode.


#### Edit bookmark

`E` in normal mode.


#### Delete bookmark

`D` in normal mode.
