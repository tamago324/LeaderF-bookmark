# LeaderF-bookmark

This Plugin use LeaderF to navigate the bookmark.

# Usage

```
:LeaderfBookmark
```
or
```
:Leaderf bookmark
```

### Commands

Add bookmark

```
:BookmarkAdd {name} {path}
```

or

```
:BookmarkAddHere [{name}]
```


Edit bookmark

```
:BookmarkEdit {name}
```


Delete bookmark

```
:BookmarkDelete {name}
```
