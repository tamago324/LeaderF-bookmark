# LeaderF-bookmark

This Plugin use LeaderF to navigate the bookmark.

# Usage

```
:LeaderfBookmark
```
or
```
:Leaderf bookmark
```

### Mappings

| key | command           |
|-----|-------------------|
| N   | add file bookmark |
| K   | add dir bookmark  |
| E   | edit bookmark     |
| D   | delete bookmark   |
